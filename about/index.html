Design process

My design process always starts with a pencil and paper, scribbling ideas and concepts, thinking about the problem and the solution.

Once I have an idea of what I want to achieve, I’ll then sit down at my computer and start with wire frames and low level mockups. Then add the imagery, typography design and details.

I am currently learning how to use Photoshop to create layouts and design.

Development process

For development, I write HTML5 and CSS3. I use SASS - a CSS extension. For those interested, the text editor I use is Atom, I use it along side Middleman, Ruby on Rails and Jekyll to build sites locally. These are the basic tools I use on every job and are my daily set up.

When time permits I would like to create Jekyll themes to earn a bit of extra money and keep my skills fresh.

I’m currently available for work and taking on new projects, so please contact me.
